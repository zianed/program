# libc (C Standard library)

C标准库

+ 标准C规范

ISO/IEC 9899 
标准C编程规范 Information technology — Programming languages — C
https://www.iso.org/standard/17782.html

伴随着POSIX规范制定

+ 内容
C11上有29个头文件。
每个头文件包含了宏、变量声明、函数。
对应的实现
