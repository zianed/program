## 自由软件
自由软件基金会Free Software Foundation (FSF)
https://www.fsf.org/


自由软件定义
https://www.gnu.org/philosophy/free-sw.html

自由软件的用户拥有四项基本自由：
+ (0)自由运行软件、
+ (1)自由学习和修改软件源代码、
+ (2)自由再发布软件拷贝、
+ (3)自由发布修改后的软件版本。


Linux Foundation
https://www.linuxfoundation.org/
